# BERT vs GPT

**Q:** When should I use [[BERT]] vs [[GPT]]?

- **[[BERT]]**: understanding tasks — classification, NER, QA. Bidirectional context.
- **[[GPT]]**: generation tasks — text completion, dialogue, code generation. Left-to-right.
- For most modern tasks, decoder-only ([[GPT]]-style) models dominate because they scale better.
