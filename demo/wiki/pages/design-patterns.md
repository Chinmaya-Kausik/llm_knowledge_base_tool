# Software Design Patterns

Reusable solutions to common software design problems.

## Patterns Used in This Vault
- **MCP Server**: each tool is a pure function (command pattern)
- **Observer**: file watchers for change detection
- **Strategy**: different layout algorithms (force-directed, CoLa, grid)
- **Facade**: the web API wraps complex tool logic behind simple endpoints
