# BERT

Bidirectional Encoder Representations from Transformers.

Pre-trained on masked language modeling + next sentence prediction.
Encoder-only [[Transformer Architecture]] with [[Self-Attention]].

## Applications
Question answering, NER, sentiment analysis, text classification.
