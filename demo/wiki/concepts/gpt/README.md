# GPT

Generative Pre-trained Transformer. Decoder-only [[Transformer Architecture]].

Scale + next-token prediction = emergent capabilities.
GPT-1 (117M) → GPT-2 (1.5B) → GPT-3 (175B) → GPT-4

Comparison with [[BERT]]: GPT is left-to-right (generation), BERT is bidirectional (understanding).
