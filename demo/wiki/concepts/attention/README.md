# Attention Mechanisms

Attention allows a model to focus on relevant parts of the input when producing output.

## Key Variants
- [[Self-Attention]] — each position attends to all positions
- [[Multi-Head Attention]] — parallel attention heads
- Cross-Attention — attends across two sequences

## History
Introduced by Bahdanau et al. (2014). Became the core of the [[Transformer Architecture]].
